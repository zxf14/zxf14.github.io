linux程序设计

  • 分区的组织形式
    • MBR：4T的限制
      • 主分区
        • hda512个字节
        • 系统盘用主分区
      • 逻辑分区
        • 每个主分区最多能分4个逻辑分区
    • EFI
    • GPT：比较新
      • GUID partition Table
  • 分区的文件系统：分区内文件数据存储的方式
    • 由操作系统制作方制定
    • FAT32：不能超过4g
    • JFS
    • NTFS
    • ext 4
  • 文件类型
    • 普通文件
    • （c开头）字符型设备驱动：数据流量小（不在硬盘上，读写是系统管理的设备）
    • b块型设备驱动：硬盘，数据量大
    • socket（读写网络发送）
    • l符号链接（相当于快捷方式，通过某一个文件名访问另一个文件）
    • （d开头）表示目录
    • ls -l列出文件详细信息
    • ls -a显示隐藏文件
  • 目录结构
    • etc目录下：配置文件
    • bin目录：应用

  • 命令

    • p w d：print work directory
    • mkdir：make directory
    • rmdir：remove directory
    • -l（列出详细信息） -R（递归） -a（显示隐藏）
    • touch：改变访问时间，希望强制编译没有改过的文件
    • cp：copy file
    • rm：remove files（也可以删文件夹，-R）
    • mv：move and rename files
    • cat：print file contents
    • ln：link file创建一个文件的快捷方式
    • more／less：一页一页的展示文件
    • 重定向<, >, >>, 2>, &>
    • 管道 | ：前一个命令的标准输出，作为后一个命令的标准输入
    • xargs 把标准输入内的文件打开，再查找
    • PATH

  • 文件类型

    • 查看
      • file 文件名
      • ls -l
    • 类型
      • 普通文件
      • 目录文件：drwxr
      • 设备文件
        • 块设备：磁盘
          • brw
          • service kuzdu start系统自动配置相应设备
        • 字符设备文件
          • crw
          • service kuzdu start自动增加删除修改字符设备文件
        • 管道文件
        • 链接文件
          • lrw
          • 软链接
            • 符号链接
            • ln –s a b
          • 硬链接（ln -s 文件 链接文件）：相当于使用了别名
            • 源文件删除后仍然存在且保留内容，作为一个普通文件
            • inode号：硬链接相同；软链接不同
            • 硬链接链接数：大于1
            • 不能给目录创建链接
            • 只有在同一文件系统中的文件之间才能创建链接
  • 文件指令
    • pwd绝对路径
    • ls：
      • 隐藏文件：以 . 开头
      • 查看隐藏文件 ls -a
      • ls -i 查看inode号
      • ls -l
        • 类型 连接数 黄建人 时间 文件／目录名 权限
  • touch创建空文件，更改文件或者目录时间
  • mkdir创建目录
    • mkdir -m 777 newdir2权限为777
  • rmdir删除空目录
  • dd创建有大小的文件，备份一段连续的空间，把iso文件写到u盘
  • cp：拷贝文件／目录（-r 递归的）
    • cp -r /boot /root
    • cp /etc/named.conf /root/named.c（改名）
  • mv：
    • mv /root/picture /root/mypicture（改名）
  • rm：删除文件
    • rm -rf /root/ab
  • wc：统计字数
    • wc –lwc file1 file2
    • -l行数
    • w单词数c字符数

  • 文本显示和处理

    • cat：显示文本文件内容，把几个文件内容附加到另一个文件中
      • cat ／etc／index
        • cat file1 >file2
      • more
        • more +10 file：从第10 行开始显示
        • more -10 file：一次显示10行
      • less
      • head
        • head -3 文件名
        • -v显示文件名
      • tail
        • 默认从倒数第10行开始显示
        • tail -f：实时的得到新追加到文件中的信息，常用来跟踪日志文件
      • sort
      • uniq
      • cut

  • comm

  • diff
    • diff -u 文件名 文件名
  • cut 命令从文件的每一行剪切字节、字符和字段并将这些字节、字符和字段写至标准输出。
  • patch
  • gerp：查找符合条件的字符串
    • grep ‘字符串’ 文件名
    • grep ‘test’ d*以d开头的文件名
    • rpm -a会把安装的所有软件显示出来rpm -a | grep kerne叫kernel的软件
  • find：效率低，可以在根，root下查找，指定查找的位置：查找文件名，按照时间
    • find 位置 -选项 文件
    • find 。 -ctime -10：十天内创建的文件
    • find 。 -name filename：按照名字搜索
  • locate：做索引
  • whereis命令位置
  • whatis命令解释
  • which：根whereis差不多

  • root用户
  • 虚拟用户：没有登陆系统的能力，但是是运行需要的，比如bin
